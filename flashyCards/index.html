JavaScript (JS)

A programming language designed to add functionality to webpages.

Operators

A function used to perform a specific operation on a value or variable. (often used mathmatically or for logic)

Variables

a labeled bucket, in which data is stored. In most cases the contents of the bucket can be dumped out, or removed; however, in the case of constants (const) these become fixed as soon as they're declared.

Variable naming conventions

There are more naming convantions than there are named people living on earth. The main two are UpperCamelCase and lowerCamelCase (as depicted respecively)

Functions

A list of instructions for the computer to follow. They may include parameters and accept arguments when they are in use.

Methods

Functions inside of Objects

Arguments

Data passed into Fucntions

Parameters

The names given to expected values

Return

data sent back from the function after it runs

Chaining methods

Putting the call for a method inplace of an argument when calling another method, so that the output of the first method becomes an argument for the second.

Strings

A string is a datatype for words and sentences (tecnically an array of characters.)

Booleans

A datatype that returns either of two values: true or false.

Undefined

undefined is a property of the global object. That is, it is a variable in global scope. ... A variable that has not been assigned a value is of type undefined . A method or statement also returns undefined if the variable that is being evaluated does not have an assigned value.

NaN

A property of the global object. In other words, it is a variable in global scope.

Null

The value null represents the intentional absence of any object value.

Alert

Used in javascript in conjunction with an argumnet to display a string.

Comments

a programmer-readable explanation or annotation in the source code of a computer program.

jQuery

jQuery is an open-sourced JavaScript library that simplifies creation and navigation of web applications.

Attributes

a piece of information which determines the properties of a field or tag in a database or a string of characters in a display.
